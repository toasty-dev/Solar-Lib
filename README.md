# What is Solar Lib?
Solar is a roblox UI Library that changes the game with many amazing features, such as, modules for config saving and themes, discord autojoining, and a very sexy syntax using a tree.

# Features
Discord Autojoining,
Tree based syntax,
Config modules,
Theme module,
